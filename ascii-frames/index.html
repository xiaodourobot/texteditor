What is Ascii?


Ascii code as it’s called has been around for quite some time, in fact, way before the internet! Did you know that originally Ascii was only capital letters and numbers? However, fast-forward to 1967, lower case and a select few characters were added, forming US-Ascii.


So with only 128 characters in use, this was all that was needed to write in English language Ascii.


Today, there are well over 258 characters that form the Ascii code we know and love so much, that help us create so many diverse and creative Ascii character frames.


Fun fact: Almost all computer systems today use Ascii code to represent character and text formats!
