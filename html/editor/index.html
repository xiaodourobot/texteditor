HTML Editor and Cleaner Online ✎ texteditor.luhui.net

How to use TextEditor?
Paste your document or start typing and formatting the text here. Activate the HTML editor feature and use the draft sheet if you need it. Download your document as Doc, PDF, HTML or any other file when you're done.

Use the demo text to experiment with the features

Text Editor Features

Find a quick explanation about each panel clicking the questionmarks ? in the bottom corners.

  • Settings – customize the dashboard and its appearance.
  • Tag Manager – execute bulk operations on the tags. Replace or delete them and their attributes.
  • Cleaner – check the desired cleaning options then clean the source code with a click of a button.
  • Find & Replace – bulk replace strings that match a pattern. Do it in the whole document, the content, in tag attributes only, in class/ID names or in style="" attributes only.
  • Color Picker – pick a color from the palette and save it for later.
  • CSS Editor – style the appearance of your document. Load the stylesheet from an external source or paste the CSS in the text field.
  • Tag Filter – delete or extract all tags to the draft sheet that match a pattern. For example: delete <img> tags which don't contain "alt" attributes.
  • Attribute Filter – delete or extract tag attributes that match the described pattern. For example: Extract "href" attributes from <a> tags
  • Click And Copy – add the most frequently used code snippets to easily copy them from here.
  • Lorem Ipsum – the gibberish text generator fills the page with placeholder text.
